# Loan Prediction Classification

## Objective

Predict whether a loan applicant is likely to receive loan approval using machine learning techniques.

## Dataset

Loan Prediction Dataset

## Approach

* Loaded and inspected the dataset.
* Handled missing values.
* Encoded categorical variables.
* Visualized loan amount, education, and annual income.
* Trained a Logistic Regression model.
* Evaluated performance using accuracy score and confusion matrix.

## Results and Insights

* Missing values were handled successfully.
* Data visualization helped understand feature distributions.
* Logistic Regression achieved good classification performance.
* Confusion matrix provided detailed evaluation of predictions.

* Pandas

* Matplotlib
